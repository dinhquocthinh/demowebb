== Description ==
Th Big Shop is a beautiful and clean child theme of Big store. 
You can create websites like gadget & digital store, fashion shop, 
furniture, electronic, food, grocery, clothing etc. It has many 
sections along with Header and footer layout combinations, Color and 
Background option and other customizing options. Theme is deeply 
integrated with WooCommerce plugin to sell your products online. Check
